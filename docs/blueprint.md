# **App Name**: CheatTech

## Core Features:

- Landing Page Hero Section: Attractive headline, subheadline, and clear CTAs (Lihat Demo, Beli Sekarang) to immediately engage visitors.
- Product Detail Page with Live Preview: Dedicated page to preview the portfolio templates and its key features before purchasing.
- Checkout Process: Simplified checkout page with only essential fields (Nama, Email) for quick order placement.
- Order Submission: Submission to the Firestore database of user order on the 'Pesan Sekarang' click. Orders are displayed on admin page.
- Success Page: Confirmation page with clear instructions for the next steps after purchase.
- Firestore Product Management: Admin UI to update portfolio website template details (price, features, etc.) in Firestore, allowing dynamic content updates without code changes.
- AI-Powered Personalized Headline Tool: Use a generative AI tool to suggest alternate headlines to site administrators. Prompt the model to be concise and conversion-focused.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and modernity, suitable for a tech-focused product aimed at students.
- Background color: Light blue (#E1F5FE), a very low saturation variant of the primary, for a clean and unobtrusive backdrop.
- Accent color: A contrasting purple (#8E44AD), 30 degrees 'left' of blue on the color wheel, for calls to action and important information.
- Body and headline font: 'PT Sans' for a modern yet friendly feel that is suitable for both headlines and body text.
- Clean and minimal layout with clear sections for easy navigation and information access.
- Simple, modern icons to represent product features and benefits.
- Subtle transitions and hover effects to enhance user experience without being distracting.
